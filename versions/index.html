React Native 版本规则

开源版本的React Native原则上每月发布一个新版本。关于版本迭代的讨论请移步 react-native-releases 仓库（注意请不要在这个仓库里讨论一般的问题）。在每次发布新的稳定版本的同时， 一般还会在最新主代码分支facebook/react-native上切出一个新的测试候选版本（RC）。 这个候选版本会在这一个月中接受大家的 尝鲜测试并积极听取 描述清楚的、有建设性的意见反馈。在解决一些重要的问题后，这一候选版本就会成为新的稳定版本。

最新候选版本（未正式发布）

要了解尝试最新的变化并提供积极的意见反馈，那就来试试最新的候选版本吧。 Facebook的官方应用会积极地替大家试用新代码，甚至在还没有切出候选版时就已经应用到上线的应用中。
