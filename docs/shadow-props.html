阴影样式属性

React 实战教程 深入学习一线大厂必备前端技能，VIP 教程限时免费领取。 立即查看 >

文档#

These properties are iOS only - for similar functionality on Android, use the elevation property.

Props#

shadowColor#

设置阴影色。

类型必填平台
color否iOS

shadowOffset#

设置阴影偏移量。

类型必填平台
object: {width: number,height: number}否iOS

shadowOpacity#

设置阴影不透明度 (乘以颜色的 alpha 分量)。

类型必填平台
number否iOS

shadowRadius#

设置阴影模糊半径。

类型必填平台
number否iOS
